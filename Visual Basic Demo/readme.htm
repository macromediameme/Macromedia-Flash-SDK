Flash 3 Open Beta

Visual Basic Project Sample Release Notes

This document last updated April 10, 1998

This directory contains a sample Visual Basic Project which uses the Shockwave Flash 3 ActiveX control. You can install the ActiveX component by running FlashAXInstall.exe. You must then select the Shockwave Flash ActiveX control from the Controls tab. When the control has been selected, it will appear in the general control tools palette.
