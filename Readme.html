Flash Player SDK

 

The enclosed Flash Player SDK is intended to enable developers to incorporate Flash playback in a standalone (ie browser independent) application. There are a number of methods available to accomplish this task:

  • VisualC++ - Sample demonstrates using the Flash ActiveX control from VisualC++.
  • VisualBasic - Sample demonstrates using the Flash ActiveX control from VisualBasic.
  • Netscape Plug-in approach - Developers who require Mac support or do not wish to adopt the ActiveX component model in their application may choose to implement the client side of the Netscape plug-in architecture. They can then use any Netscape plug-in's on Mac/Win directly (including the Flash plug-in). In support of this option a developer may also freely aquire the source code for the Netscape browser. See mozilla.org for more info or proceed directly to the mozilla download.
  • Java - Sample includes source, java api documentation, and the complete set of class files for the java player.

The Flash Player SDK also includes the following:

  • A white paper describing the SWF File Format.
  • The complete SWF File Format Specification.
  • Two sample projects (swfparse and urlsubst) demonstrating how to manipulate a SWF File.
